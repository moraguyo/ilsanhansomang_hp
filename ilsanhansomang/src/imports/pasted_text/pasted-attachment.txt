첨부한 세 이미지를 참고해주세요.
1번 이미지 = 지구본 초기 상태 (작게 시작)
2번 이미지 = 지구본 확대 후 카드 등장 상태 (목표 디자인)
3번 이미지 = 현재 피그마 구현 화면 (잘못된 결과물)

아래 내용을 기준으로 MISSION 섹션을 전면 수정해주세요.

## ❌ 문제 1: 지구본 줌인 애니메이션이 없음

**현재 문제:**
지구본이 처음부터 크게 고정되어 있고
스크롤에 따른 줌인 효과가 전혀 없습니다.

**수정 사항:**
- 섹션 진입 시 지구본이 작은 크기(scale 0.3 정도)로 화면 중앙에 위치합니다
- 사용자가 스크롤을 내리면 지구본이 점점 커지면서(scale 1.5~2.0)
  마치 앞으로 다가오는 것처럼 보여야 합니다
- 지구본이 일정 크기 이상 커지면 화면 하단으로 밀려나가듯 자리를 잡고
  그 위로 MISSION 타이틀과 카드들이 자연스럽게 나타납니다
- 전체 효과는 sticky 방식으로 구현해주세요
  (스크롤하는 동안 지구본 확대 애니메이션이 진행되도록)

애니메이션 타임라인:
스크롤 0%   → 지구본 scale(0.3), 화면 중앙 상단 위치, 카드 보이지 않음
스크롤 50%  → 지구본 scale(1.0), 중앙에서 점점 커짐
스크롤 80%  → 지구본 scale(1.8), 화면 하단으로 이동 시작
             → MISSION 타이틀 + 카테고리 탭 + 카드 슬라이더 나타남
스크롤 100% → 완전한 MISSION 섹션 화면으로 전환

## ❌ 문제 2: 카드 3D 입체 효과 없음

**현재 문제:**
카드가 모두 정면으로만 보여서 평평하고 단조롭습니다.

**수정 사항:**
- 가운데 카드: 정면으로 크게 표시 (직사각형, 세로형)
- 좌측 카드: rotateY(15deg) 정도로 오른쪽으로 기울여서
  왼쪽 면이 뒤로 들어가 보이는 입체감
- 우측 카드: rotateY(-15deg) 정도로 왼쪽으로 기울여서
  오른쪽 면이 뒤로 들어가 보이는 입체감
- 양옆 카드는 가운데 카드보다 살짝 작게(scale 0.85 정도) 표시
- perspective를 적용해서 실제 3D처럼 보이게 해주세요
- 카드 전환 시 회전하면서 자연스럽게 넘어가는 효과

---

## ❌ 문제 3: 카드 디자인이 시안과 다름

**현재 문제:**
카드 내부 디자인이 시안(2번 이미지)과 다릅니다.

**수정 사항 - 카드 디자인 (2번 시안 이미지 기준):**
- 카드 형태: 세로형 직사각형 (portrait 비율)
- 카드 내부 구조:
  상단 2/3: 배경 이미지 (풀 커버)
  하단 1/3: 어두운 반투명 오버레이 위에 텍스트
  - 한글 사역명 (굵고 크게, 흰색)
  - 영문명 (작게, 흰색)
  - 우측 하단: 원형 화살표 버튼 (→)
- 카드 모서리: 부드러운 border-radius 적용
- 카드 배경색: 각 사역별 이미지 사용

---

## ❌ 문제 4: 화살표 네비게이션 위치

**현재 문제:**
화살표가 카드 위에 겹쳐 있어서 카드를 가립니다.

**수정 사항:**
- 좌측 화살표(<): 섹션 가장 왼쪽 끝에 세로 중앙 정렬
- 우측 화살표(>): 섹션 가장 오른쪽 끝에 세로 중앙 정렬
- 화살표 스타일: 원형 버튼, 반투명 배경
- 카드와 겹치지 않도록 섹션 양 끝에 고정

---

## ❌ 문제 5: 카테고리 탭 메뉴 위치

**현재 문제:**
탭 메뉴가 MISSION 타이틀 오른쪽에 있는데
시안에서는 타이틀과 카드 사이에 있어야 합니다.

**수정 사항:**
- MISSION 타이틀은 좌측에 크게
- 카테고리 탭(국내선교, 해외선교, 파송선교사소식, 중보기도, 디아코니아)은
  타이틀 우측 또는 타이틀 아래 우측 정렬로 배치
- 선택된 탭은 어두운 배경 + 흰 글씨로 활성화 표시
- 탭 클릭 시 해당 카테고리 카드로 슬라이더 이동

---

## ✅ 수정 사항 요약

1. 지구본 줌인 효과: 스크롤에 따라 작게→크게 확대되면서
   카드 섹션이 자연스럽게 등장 (sticky 스크롤)
2. 카드 3D 입체감: 가운데 정면, 양옆 15도 기울기 + perspective 적용
3. 카드 디자인: 세로형, 상단 이미지 + 하단 텍스트 + 원형 화살표 버튼
4. 화살표 위치: 섹션 양쪽 끝으로 이동, 카드와 겹치지 않게
5. 탭 메뉴: MISSION 타이틀과 카드 슬라이더 사이에 배치

섹션 배경색(흰색), 폰트, 텍스트 내용은 변경하지 마세요.
반드시 2번 시안 이미지를 기준으로 수정해주세요.
