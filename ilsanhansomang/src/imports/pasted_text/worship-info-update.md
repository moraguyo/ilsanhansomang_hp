첨부한 두 이미지를 참고해주세요.
1번 이미지 = 원본 시안 (이렇게 만들어야 함)
2번 이미지 = 현재 구현 화면 (잘못된 결과물)

아래 내용을 기준으로 WORSHIP INFO 섹션을 전면 수정해주세요.

## ❌ 문제 1: 장년예배 표 구조가 다름

**현재 문제:**
주일예배 시간이 한 줄로 나열되어 있고
표의 행 구분과 세로선이 없어서 시안과 전혀 다른 모양입니다.

**수정 사항 - 장년예배 표 정확한 구조:**

표는 아래처럼 세로 구분선(|)을 기준으로 
왼쪽에 항목명, 오른쪽에 시간과 장소가 나뉘어야 합니다.

장년예배

주일예배     |   1부          오전 07:30
             |   2부          오전 09:30      한소망채플(4층)
             |   3부          오전 11:30
             |   4부(청년)    낮 02:00

말씀사경회   |   오전 07:30                   한소망채플(4층)

한밤의 기도회|   오전 07:30                   한소망채플(4층)

매일 기도회  |   오전 07:30                   한소망채플(4층)

월삭 온가족  |   오전 07:30                   한소망채플(4층)
한밤의 기도회|

- 각 행 사이에 얇은 가로 구분선을 넣어주세요
- 항목명 열과 시간 열 사이에 얇은 세로 구분선(|)을 넣어주세요
- 주일예배는 1부~4부가 세로로 나열되어야 합니다
- 폰트는 Wanted Sans, 색상은 시안과 동일하게 유지해주세요
- 표 전체에 테두리 박스는 없고 내부 선만 있는 심플한 라인 테이블입니다

## ❌ 문제 2: 다음세대 예배 탭 메뉴가 없음

**현재 문제:**
다음세대 예배 섹션에서 탭 메뉴가 완전히 사라졌습니다.

**수정 사항 - 탭 메뉴 구조 복원:**

표 상단 또는 좌측에 아래 탭 메뉴를 작은 글씨로 넣어주세요.

탭 목록 (좌측 세로 또는 상단 가로 배치):
• 해피(영아, 유아, 유치)   ← 기본 선택 탭
• 드림(유년, 초등)
• 영이에베(리레나)
• 청소년(중등, 고등)
• 소망부(어린이, 소망부)

- 선택된 탭은 색상 강조 또는 밑줄로 활성화 표시
- 탭을 클릭하면 해당 탭의 예배 시간표가 표시됩니다
- 1번 시안 이미지의 좌측 탭 메뉴 스타일을 참고해주세요

## ❌ 문제 3: 탭별 표 내용 통일

**수정 사항:**
모든 탭(해피/드림/영이에베/청소년/소망부)을 클릭했을 때
표시되는 내용은 아래 해피 탭 내용으로 통일해주세요.

표 구조:

다음세대 예배

해피영아부 (1-3세)  |  1부 09:30   3층 환영해방
                    |  2부 11:30

해피유아부 (4-5세)  |  1부 09:30   3층 행복해방
                    |  2부 11:30

해피유치부 (6-7세)  |  1부 09:30   3층 사랑해방
                    |  2부 11:30

- 장년예배 표와 동일한 스타일의 라인 테이블로 만들어주세요
- 각 부서명과 시간 사이에 세로 구분선(|) 넣어주세요
- 행 사이에 얇은 가로 구분선 넣어주세요

## ❌ 문제 4: WORSHIP INFO 타이틀 폰트

**현재 문제:**
WORSHIP INFO 타이틀이 산세리프 볼드체로 되어 있음

**수정 사항:**
- "WORSHIP INFO" 타이틀을 세련된 영문 세리프체로 변경해주세요
- MESSAGE 섹션 타이틀과 동일한 세리프 폰트 적용

## ✅ 수정 사항 요약

1. 장년예배 표: 세로 구분선 + 가로 구분선 있는 라인 테이블로 재구성
   주일예배 1부~4부 세로 나열, 각 항목별 시간/장소 정렬
2. 다음세대 탭 메뉴 복원: 해피/드림/영이에베/청소년/소망부 5개 탭
3. 탭 내용: 모든 탭 클릭 시 해피영아부~유치부 표 내용으로 통일
4. WORSHIP INFO 타이틀: 세리프체로 변경
5. 전체 레이아웃(좌측 사진 + 우측 표, 하단 좌측 표 + 우측 사진)은 유지

텍스트 내용, 배경색, 전체 섹션 구조는 변경하지 마세요.
반드시 1번 시안 이미지를 기준으로 수정해주세요.
